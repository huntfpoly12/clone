# Jangbuda Vue App

## Project setup
```
npm install or yarn
```

### To run the test under local
```
npm run serve
```
or
```
yarn serve
```

### Compiles and minifies for development
```
npm run build
```
or
```
yarn build
```

### Compiles and minifies for production
```
npm run build:prod
```
or
```
yarn build:prod
```

### Lints and fixes files
```
npm run lint or yarn lint
```
